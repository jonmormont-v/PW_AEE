A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, o que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Alguns dos recursos de linguagem HTML vistos até o momento:

Recursos Tags associadas Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar listas de itens nao ordenados
Imagens <img> Inserir imagens
